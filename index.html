04 — Selected work

Things I've built.

01 / GENAI
Chrome Extension · FastAPI · LangChain · OpenAI · FAISS

YouTube Video Q&A — a RAG-powered Chrome extension.

A GenAI Chrome Extension enabling natural-language Q&A over any YouTube video. A RAG pipeline does semantic chunking of the transcript, indexes it in FAISS, and uses GPT-4o-mini for grounded response generation — with timestamp-aware answers you can click to jump into the video.

A dual-pipeline architecture separates retrieval-based Q&A from full-transcript summarization. FastAPI orchestrates LangChain runnables to handle retrieval, context injection, and generation — with strict context grounding and caching to cut hallucinations.

Pipeline: RAG + FAISS
Response: timestamped
Grounding: strict + cached

02 / RESEARCH
PyTorch · Hugging Face · PEFT · 3B LLM

Parameter-Efficient Fine-Tuning of Large Language Models.

Fine-tuned a 3B-parameter LLM on the EmpatheticDialogues dataset (25K+ conversations) using LoRA, Adapters, and Prompt/Prefix Tuning — benchmarked rigorously against full fine-tuning baselines.

Identified LoRA configurations that match full fine-tuning performance with ~95% fewer trainable parameters, massively reducing GPU memory and inference cost. Built reproducible PyTorch training/eval pipelines and improved emotion classification through prompt-based task decomposition and chain-of-thought prompting.

Params trained: -95%
Model size: 3B
Data: 25K+ dialogues

Python · Scikit-learn · Streamlit

Crime Rate Prediction & Analysis — end-to-end ML dashboard.
